2422-[电气] 欠压
======================

符合预期，用例后补。

-----------

用例依赖: 0

概述: 采集器合闸状态，进线电压低于177v，且持续2秒判定为 "欠压"。持续时长多余3秒后，执行分闸操作。电压恢复2秒后将消警，持续10分钟后将执行合闸操作。

    * 参考\ :ref:`典型安装拓扑`, 搭建硬件环境。
